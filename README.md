# Learning Web Dev

This repository is a work in progress while I learn web development and build small projects for fun.

Right now, the main project is a deliberately awful birthday selector. The goal is to make choosing a date much harder than it should be, so the bad user experience is intentional. The project is not finished yet, and I expect it to change as I keep learning.

## Current Project

The current app is a simple HTML/CSS/JavaScript page that lets you pick a birthday using absurdly large slider ranges.

## Why This Repo Exists

I am an aspiring software engineer using this repo to practice:

- HTML
- CSS
- JavaScript
- DOM manipulation
- Building small projects from scratch
